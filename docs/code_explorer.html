Potential Issues
WARNQuadratic Threat Scan
Each square scans all others every frame so complexity is O n squared and may slow down as entity count grows.
Basics:More squares means much more work.
Engineering Insight:Frame time can spike on larger scenes.
Architecture Insight:Spatial partitioning can cap neighbor checks.
Improvement: add a uniform grid or quadtree for nearby lookup.
Quadtree concept
WARNHardcoded Behavior Values
Flee radius and strength are inline values in update logic so tuning and testing behavior requires code edits.
Basics:Numbers appear without named meaning.
Engineering Insight:Tuning is harder and less reproducible.
Architecture Insight:Config constants should sit at module boundary.
Improvement: expose behavior constants as top level settings.
Replace magic values
